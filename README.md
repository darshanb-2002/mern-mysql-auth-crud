# MERN Auth Dashboard Project

## Features
- User Registration
- User Login (JWT Authentication)
- Forgot & Reset Password
- Add Items (CRUD)
- Protected Routes

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MySQL

## How to Run

### Backend
cd backend
npm install
npx nodemon server.js

### Frontend
cd frontend
npm install
npm run dev
